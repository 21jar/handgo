### 本地开发运行部署

* 下载 zip 直接解压或安装 git 后执行克隆命令 `git@github.com:imfing/handgo.git`

* 在 MySQL 中创建数据库 `handgo`

* 导入项目至 IDEA，将工程中的 `handgo.sql`导入 MySQL

* 运行
